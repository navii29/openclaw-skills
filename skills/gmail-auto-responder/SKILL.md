# Skill: Gmail Auto-Label + Smart Replies

## Use Case
Deutsche Unternehmen erhalten täglich 50-200 Emails. Dieser Skill klassifiziert automatisch, labelt und generiert Antwort-Vorschläge – spart 2-3 Stunden pro Tag.

## Problem
- Emails manuell sortieren ist zeitaufwendig
- Wichtige Kundenanfragen gehen unter
- Gleiche Anfragetypen werden immer wieder neu beantwortet

## Lösung
KI-basierte Email-Verarbeitung:
1. Neue Emails abrufen (IMAP)
2. Inhalt klassifizieren (Rechnung, Support, Anfrage, Spam)
3. Gmail-Label automatisch setzen
4. Antwort-Vorschlag generieren (basierend auf Templates)

## Inputs
- Gmail IMAP/SMTP Zugangsdaten
- Label-Kategorien (konfigurierbar)
- Antwort-Templates pro Kategorie

## Outputs
- Auto-Labeling in Gmail
- Antwort-Vorschläge als Drafts
- CSV-Report mit Klassifizierungen

## API Keys Required
- Gmail App Password

## Setup Time
10 Minuten

## Use Cases
- Support-Ticket Klassifizierung
- Angebotsanfragen erkennen
- Rechnungsanfragen priorisieren
- Spam/Marketing filtern
